# Bullox - Complete Frontend

A modern fintech trading platform built with React, TypeScript, and Tailwind CSS.

## Features
- 11 Complete Pages
- Light theme with colorful gradients
- Real-time quotes and charts
- Portfolio analytics
- Copy trading system

## Tech Stack
- React 18 with Vite
- TypeScript
- Tailwind CSS
- React Router v6
- TanStack Query
- Zustand for state management

## Pages
1. Dashboard - Portfolio overview
2. Market - Live quotes and candles
3. News - AI sentiment analysis
4. Calendar - Economic events
5. Watchlist - Symbol tracking
6. Alerts - Price alerts
7. Copy Trading - Trader profiles
8. Settings - Account management
9. Billing - Plans and invoices
10. Analytics - Performance metrics
11. Trader Detail - Trader profiles
